## Overview

Spearheaded the backend integration between Domino’s and Zomato IRCTC Train Orders using a dynamic template-based architecture. This extended Domino's ordering reach to India's vast rail passenger network.

## Key Features

- **Third-Party Extensibility**: Optimized API for seamless integration with external partners.
- **High Reliability Pipeline**: Redesigned end-to-end order processing pipelines to reduce drop-offs in low-connectivity environments.

## Impact

- Boosted IRCTC flow conversion rate by **230 bps**.
- Grew train order volume by **25%**.
