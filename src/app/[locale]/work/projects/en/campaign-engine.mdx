## Overview

Architected a zero-deployment, multi-tenant Campaign Engine microservice with a custom rule evaluation core. This system processes complex condition schemas to enable real-time cart personalization and dynamic promotions for millions of daily user sessions.

## Key Features

- **Zero-Deployment Configs**: Allows marketing teams to launch or change campaigns instantly without code deployments.
- **Custom Rule Engine**: Evaluates complex logic against real-time cart states.
- **Multi-Tenant Architecture**: Supports diverse campaign rules across multiple brands natively.

## Impact

- Significantly reduced time-to-market for promotional campaigns.
- Empowered marketing teams with real-time A/B testing and experimentation capabilities.
